# Git 기초

## 개념

### SCM이란?

sorce code manager 의 약자로, 코드의 버전을 관리하기 위해 존재한다.

### Git

git은 Linus Torvalds가 만든 scm으로, 현제 가장 많이 사용된다.

### Github

github는 git이 관리하는 repo의원격 저장소를 제공하는서비스다.가장 많이 사용하며, 오픈소스 프로젝트를 이 성장하는 곳이다. 최근 MS에 인수 되었다.
